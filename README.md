# JavaScript30

30 day vanilla js coding challenge

## Run Locally

Clone the project

```bash
git clone https://github.com/isaaccolls/wesbosJavaScript30.git
```

Go to the project directory

```bash
cd wesbosJavaScript30
```

Start the some server like

```bash
python3 -m http.server 8000
```
